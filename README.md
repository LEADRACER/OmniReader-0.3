# OmniReader 🌌

A premium, universal document reader built with modern web technologies. Experience documents in a whole new dimension with smart summarization, interactive Q&A, and instantaneous translation.

![OmniReader Hero](https://images.unsplash.com/photo-1544391496-1ca7c9765751?auto=format&fit=crop&q=80&w=1200)

## ✨ Features

- **🚀 Multi-Format Support**: Seamlessly read PDF, Markdown (`.md`), and Plain Text (`.txt`) files.
- **🎙️ Voice Over**: Integrated Text-to-Speech (TTS) engine to listen to your documents.
- **🌍 Universal Translation**: Simulated translation across multiple languages (Spanish, French, Japanese, German, Chinese).
- **🧠 Smart Reading Modes**:
  - **Detailed**: The full experience with original formatting.
  - **Summary**: AI-powered (simulated) core theme extraction.
  - **Q&A**: Interactive query engine for document insights.
- **💎 Premium Design**: Glassmorphism UI, smooth GSAP animations, and a dynamic nebula background.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Animations**: [GSAP](https://greensock.com/gsap/)
- **Icons**: [Ionicons](https://ionicons.com/)
- **Parsing**: [PDF.js](https://mozilla.github.io/pdf.js/), [Marked.js](https://marked.js.org/)

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/LEADRACER/OmniReader-0.3
   ```
2. Open `index.html` in your favorite modern browser.
3. Start uploading and reading!
